# Landing Page (Homepage)

This folder contains a standalone, static landing page homepage (HTML/CSS/JS) styled in a clean, modern marketing style.

## Run locally

Option A (simple): open `index.html` in your browser.

Option B (recommended): serve locally so relative assets load consistently.

```bash
cd landing_page
python3 -m http.server 8080
# then open http://localhost:8080
```

## Structure

- `index.html` — homepage
- `styles.css` — styling
- `main.js` — minimal interactions
- `assets/` — images extracted from the provided PDF deck
